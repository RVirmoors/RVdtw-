template<class T>
class CoreFrequencyDomainFeatures< T >

Instantiations of the class should be of either 'float' or 'double' types and no others

T CoreFrequencyDomainFeatures< T >::spectralCentroid ( std::vector< T >  magnitudeSpectrum)

calculates the spectral centroid given the first half of the magnitude spectrum of an audio signal.

Do not pass the whole (i.e. mirrored) magnitude spectrum into this function or you will always get the middle index as the spectral centroid
