void Chromagram::setChromaCalculationInterval ( int  numSamples)

Set the interval at which the chromagram is calculated. As the algorithm requires a significant amount of audio to be accumulated, it may be desirable to have the algorithm not calculate the chromagram at every new audio frame. This function allows you to set the interval at which the chromagram will be calculated, specified in the number of samples at the audio sampling frequency

Parameters
numSamplesthe number of samples that the algorithm will receive before calculating a new chromagram
